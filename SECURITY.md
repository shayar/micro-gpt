# Security Policy

## Supported status

MicroGPT is currently in early development.

| Version | Status |
|---|---|
| `main` | Active development |
| Public release | Not yet available |

## Security posture

MicroGPT is not approved for public production use yet.

The project uses:

- Secure login before chat access
- JWT-based authentication for the starter
- Maturity gate before broader access
- Basic safety checks before model inference
- Safety and licensing review before public release

## Reporting a vulnerability

Please do not open a public GitHub issue for security vulnerabilities.

Report security concerns privately to:

```txt
security@example.com
```

Replace this email before publishing the repository.

Include:

- Description of the issue
- Steps to reproduce
- Impact
- Affected files or endpoints
- Suggested fix, if known

## AI safety issues

Please also report:

- Authentication bypasses
- Maturity gate bypasses
- Prompt-injection bypasses
- Unsafe violent content generation
- Sexual or explicit content generation
- Non-consensual media generation
- Data leakage
- Model/license misuse

## Public access policy

Public access must remain disabled until:

- Security review is complete
- Safety review is complete
- License review is complete
- Evaluation report is complete
- Maintainers approve release readiness
