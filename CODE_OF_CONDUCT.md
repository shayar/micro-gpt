# Code of Conduct

## Our pledge

We are committed to making MicroGPT a welcoming, respectful, and productive open-source community.

We welcome contributors regardless of experience level, background, identity, location, or skill level.

## Expected behavior

Contributors are expected to:

- Be respectful and constructive.
- Give and receive feedback professionally.
- Focus on the technical work and project goals.
- Respect safety, security, and licensing requirements.
- Help create a learning-friendly environment.

## Unacceptable behavior

The following behavior is not acceptable:

- Harassment, insults, or personal attacks
- Discriminatory language
- Threats or intimidation
- Sharing private information without permission
- Submitting unsafe, malicious, or intentionally harmful code
- Attempting to bypass safety controls

## AI safety expectations

Because MicroGPT is an AI project, contributors must not intentionally submit features that enable:

- Violent content generation
- Sexual or explicit content generation
- Non-consensual media creation
- Voice cloning without consent
- Deepfake or impersonation misuse
- Prompt-injection bypasses
- Hidden data exfiltration

## Enforcement

Maintainers may remove comments, close issues, reject pull requests, or restrict participation when behavior violates this code of conduct.

## Reporting

Report concerns privately through the security contact listed in `SECURITY.md`.
