# Phase 1 Safety Gate

The Phase 1 safety system is intentionally simple and conservative.

It blocks obvious examples of:

- Weaponized/violent instructions
- Explicit sexual content requests
- Attempts to bypass safety or reveal hidden system instructions

This is **not** enough for public release.

Before public release:

- Add open-source classifier evaluation
- Add red-team prompt set
- Add output safety tests
- Add tool permission boundaries
- Add human review workflow for safety incidents
