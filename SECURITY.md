# Security Policy

## Current maturity

This repository starts at `research` maturity.

Public access must remain disabled until security, safety, evaluation, and license gates pass.

## Do not commit

- `.env`
- passwords
- JWT secrets
- model weights
- user data
- private documents
- local vault contents
- runtime logs with sensitive content

## Reporting issues

For now, open a private issue or contact the project maintainer directly. Public security disclosure process should be added before private beta.
